# Awesome Stuff :sunglasses:

## "Awesome" on GitHub

Definitely check out the
[sindresorhus/awesome](https://github.com/sindresorhus/awesome) repo on GitHub
to see some cool stuff. Just about any topic is in there.

They have links to a bunch of other lists that great for learning things like:

- The programming languages the kids are using these days
- Machine learning tutorials
- Good books to get
- Game development
- Miscellaneous stuff to make software development easier (like editors and
    the different kind of terminal shells you can use)
- Tips on how to Git and GitHub
- Game development
- A ton of other things

There's something on almost any topic and it's definitely worth a good look!

## Free Programming Books

Check out the [EbookFoundation/free-programming-books](
    https://github.com/EbookFoundation/free-programming-books)
GitHub repo for a massive list of great publicly available programming books.

Nearly all the programming languages can be found on there.
